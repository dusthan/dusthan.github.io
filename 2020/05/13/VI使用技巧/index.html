VI使用技巧

Posted on 2020-05-13

VI 使用技巧

快捷键

多文件替换

  • 把需要进行操作的文件进行标记

    :args ./**/*.py

    **/ 表示遍历子目录，以上命令表示 当前目录及子目录下所有的python文件

  • 将所有标记过的文件执行动作

    :argdo %s/print/Print/g | update

多文件查找

在当前打开文件中查找 :vimgrep /pattern/ %

在当前目录下查找所有 :vimgrep /pattern/ *

在当前目录及子目录下查找所有 :vimgrep /pattern/ **

查找当前目录下所有.c文件 :vimgrep /pattern/ *.c

只查找子目录 :vimgrep /pattern/ */**

移动到当前位置的后10个 :cn10

移动到当前位置的前20个 :cp20

定位到第一个 :cfirst

定位到最后一个 :clast

打开Quickfix :cw/copen

关闭quickfix :ccl/cclose

lvim[grep] 与 vim[grep] 的区别

  • lvim 使用窗口的本地列表，vim 使用 quickfix 列表。
  • lvim 的窗口本地列表跳转命令为：:lnext 、 :lprev 、 :lfirst 及 :llast

VIM 光标移动

字符移动

  • h
  • j
  • k
  • l
  • i

单词移动

  • w
  • b
  • e
  • W
  • B
  • E
  • $
  • ^
4w 向后移动4个单词
^y$ 拷贝一行
yy 拷贝一行
p paste

相对屏幕移动

  • C+f 向下翻页
  • C+b 向上翻页
  • C+e 逐行下滚
  • C+y 逐行上滚
  • H 光标移到屏幕首行
  • L 光标移到屏幕尾行
  • M 光标移到屏幕中间行
  • zt 置顶当前行，滚动使当前光标位置位于屏幕首行
  • zz 滚动使当前光标位置位于屏幕中间行
  • zb 滚动使当前光标位置位于屏幕尾行

文件中移动

  • :set number 显示行号
  • :10 移动光标到第10行
  • gg 移动光标到文件头
  • G 移动光标到文件尾
  • ggyG 拷贝整个文件
  • gg"*yG 拷贝整个文件到系统剪贴板

VIM 寄存器

无名寄存器（””）

倘若我们没有指定使用寄存器， Vim 将缺省使用无名寄存器，它可以双引号表示。

例如 ""p 这个命令，他完全等同于 p 命令。

数字寄存器

行内删除（small delete）寄存器

"-，缓存行内删除内容

具名（named）寄存器

"a - "z或"A - "Z，指定时可用；

只读（read-only）寄存器

":, "., "%, "#，分别缓存最近命令、最近插入文本、当前文件名、当前交替文件名；

表达式（expression）寄存器

"=，只读，用于执行表达式命令；

选择及拖拽（selection and drop）寄存器

"*, "+, "~，存取GUI选择文本，可用于与外部应用交互，使用前提为系统剪切板（clipboard）可用；

黑洞（black hole）寄存器

"_，不缓存操作内容（干净删除）

模式寄存器（last search pattern）

"/，缓存最近的搜索模式

举例：

"byy 将当前行复制到b寄存器
"bp 粘贴b寄存器的内容
gh 进入select mode, 选择要复制的内容，Ctrl+o进入Visual mode, "+y 复制到系统剪贴板

VIM多文件

打开与关闭

vim -p test01.py test02.py sub/sub_test01.py

:tabe[dit] {file}   edit specified file in a new tab
:tabf[ind] {file}   open a new tab with filename given, searching the 'path' to find it
:tabc[lose]         close current tab
:tabc[lose] {i}     close i-th tab
:tabo[nly]          close all other tabs (show only the current tab)

移动标签

:tabs         list all tabs including their displayed window
:tabm 0       move current tab to first
:tabm         move current tab to last
:tabm {i}     move current tab to position i+1

标签跳转

:tabn         go to next tab
:tabp         go to previous tab
:tabfirst     go to first tab
:tablast      go to last tab

在正常模式下，还可以使用以下快捷键：

gt            go to next tab
gT            go to previous tab
{i}gt         go to tab in position i

可以设置更通用的切换标签页快捷键，比如

noremap <C-L> <Esc>:tabnext<CR>
noremap <C-H> <Esc>:tabprevious<CR>
