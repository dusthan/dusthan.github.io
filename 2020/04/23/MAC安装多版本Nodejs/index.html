MAC安装多版本Nodejs

Posted on 2020-04-23 | In work

环境搭建

安装n

添加n相关的环境变量

  • 添加export N_PREFIX=/Users/xxx/soft/app/node 到 ~/.bash_profile
  • source ~/.bash_profile

用n安装node

  • 安装lts版本

    n lts

  • 安装latest版本

    n latest

  • 安装指定版本（10.16.0）

    n 10.16.0

问题

安装完成后，运行 node -v 提示zsh: command not found: node

此时，运行n doctor,发现问题：'/Users/xxx/soft/app/node/bin' is not in PATH

添加export PATH=/Users/xxx/soft/app/node/bin:$PATH 到 ~/.bash_profile

source ~/.bash_profile
